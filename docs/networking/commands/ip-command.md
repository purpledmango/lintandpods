# IP Command Reference

Modern network configuration using the `ip` command from iproute2 package.

## Quick Reference

| Task | Command |
|------|---------|
| Show all interfaces | `ip link show` |
| Show IP addresses | `ip addr show` |
| Show routing table | `ip route show` |
| Show neighbors (ARP) | `ip neigh show` |

## Examples

### View Network Interfaces

```bash
ip addr show
```
